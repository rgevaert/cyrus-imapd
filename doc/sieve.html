Sieve: A mail filtering language

Sieve is a Internet standards-track protocol for filtering messages on delivery. The Cyrus Sieve implementation supports filing messages into specific folders, forwarding messages, rejecting messages, and the standard vacation function. It can reply to messages based on headers or envelope information.

Cyrus Sieve

Cyrus Sieve mostly implements RFC 3028. The latest sieve RFC is 5228. It also implements the (drafts of the) following sieve extensions:
  • RFC 3894: Copying Without Side Effects
  • RFC 5173: Body Extension
  • RFC 5230: Vacation Extension
  • RFC 5232: Imap4flags Extension
  • RFC 5231: Relational Extension
  • RFC 5233: Subaddress Extension
  • RFC 5429: Reject and Extended Reject Extensions
  • RFC 5435: Extension for Notifications
  • (? draft-murchison-sieve-regex-08) Regular Expression Extension
